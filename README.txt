Prerequisites:
Eclipse IDE (Environment variables should be set for Java)

Steps to Setup the project:
1. Download the project
2. Import this maven project in Eclipse by File > Import > Select existing maven projects > Next > Browse > select the project > click Finish

Steps to run the test:
1. Go to src/test/java > cucumberOptions
2. Right click on TestRunner.java

To view the test results report after execution:
1.Go to target folder > cucumber folder
2.Copy the location of index.html
3.Paste in your browser address bar & Enter
